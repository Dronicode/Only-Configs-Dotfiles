# Autocompletion & AI

## Blink.cmp (Autocompletion)

| Shortcut          | Mode   | Action (mnemonic)                   | Defined In                   |
| ----------------- | ------ | ----------------------------------- | ---------------------------- |
| `<C-y>`           | Insert | accept completion (\[y]es)          | `plugins/autocompletion.lua` |
| `<C-space>`       | Insert | completion menu toggle/docs         | `plugins/autocompletion.lua` |
| `<C-n>`/`<Down>`  | Insert | completion next item                | `plugins/autocompletion.lua` |
| `<C-p>`/`<Up>`    | Insert | completion \[p]revious item         | `plugins/autocompletion.lua` |
| `<C-e>`           | Insert | completion \[e]scape/hide           | `plugins/autocompletion.lua` |
| `<C-k>`           | Insert | signature help toggle (\[k]ey info) | `plugins/autocompletion.lua` |
| `<Tab>`/`<S-Tab>` | Insert | \[s]nippet jump next/previous       | `plugins/autocompletion.lua` |
